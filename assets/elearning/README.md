# Custom eLearning sample videos

Drop sample videos for the Custom eLearning page carousel here.

## Naming convention
Use lowercase, hyphenated filenames matching the sample they belong to:

- `sample-branching.mp4` — Branching scenario
- `sample-walkthrough.mp4` — Interactive walkthrough
- `sample-microlearning.mp4` — Microlearning module
- `sample-animated.mp4` — Animated explainer
- `sample-assessment.mp4` — Assessment & feedback

## Specs
- **Format:** `.mp4` (H.264) or `.webm`
- **Aspect ratio:** 16:9
- **Resolution:** 1280×720 or 1920×1080
- **Length:** 8–20 sec loops
- **Size:** ideally under 8 MB each
- Videos autoplay muted + loop when centered in the carousel.
